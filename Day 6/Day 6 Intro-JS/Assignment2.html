JavaScript Array Methods

The arr.slice(startIndex [endIndex]) Method

The slice() method returns a shallow copy of a portion of an array into a new array object selected from start to end (end not included) where start and end represent the index of items in that array. The original array will not be modified.

animals = ['ant', 'bison', 'camel', 'duck', 'elephant']

The arr.splice(startIndex, deleteCount, "new val") Method

The splice() method changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.

months = ['Jan', 'March', 'April', 'June']

The arr.reverse() Method

The reverse() method reverses an array in place. The first array element becomes the last, and the last array element becomes the first.

array = [1, 2, 3, 4]

The arr.sort() Method

The sort() method sorts the elements of an array in place and returns the sorted array. The default sort order is ascending, built upon converting the elements into strings, then comparing their sequences of UTF-16 code units values.

letters = ['D', 'W', 'G', 'A', 'V', 'H', 'S']

The arr.toString() Method

The toString() method returns a string representing the specified array and its elements.

array1 = [1, 'a', 5, '1a', 8]

The arr.toLocaleString() Method

The toLocaleString() method returns a string representing the elements of the array. The elements are converted to Strings using their toLocaleString methods and these Strings are separated by a locale-specific String (such as a comma ",")

array2 = [1, 'a', 'March']
