JavaScript Array Methods

The arr.concat(arr2) Method

The concat() method is used to merge two or more arrays:

arr = [[1, 4]]

arr2 = [2, [3]]

The arr.join() Method

The join() method creates and returns a new string by concatenating all of the elements in an array (or an array-like object), separated by commas or a specified separator string:

elements = ['My', 'Name', 'Is', 'Harshita', 'Verma']

The arr.pop() Method

The pop() method removes the last element from an array and returns that element.

coffee = ['espresso', 'latte', 'cappuccino', 'macchiato', 'americano', 'mocha', 'ristretto']

The arr.push(value) Method

The push() method adds one or more elements to the end of an array and returns the new length of the array:

coffee2 = ['espresso', 'latte', 'cappuccino', 'macchiato']

icedCoffee = ['frappuccino', 'nitro']

The arr.unshift(value) Method

The unshift() method adds one or more elements to the beginning of an array and returns the new length of the array:

arrr = ['A', 'B', 'C']

The arr.shift() Method

The shift() method removes the first element from an array and returns that removed element:

arr3 = ['A', 'B', 'C', 'D']
